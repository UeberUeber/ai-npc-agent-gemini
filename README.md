## Implementation Details (л…јл¬ё мҷё кө¬нҳ„)

л…јл¬ём—җм„ң лӢӨлЈЁм§Җ м•ҠлҠ” **мӢӨм ң кө¬нҳ„ мӢң н•„мҡ”н•ң м—Јм§Җ мјҖмқҙмҠӨ мІҳлҰ¬**мһ…лӢҲлӢӨ.

### 1. мһҗлҠ” NPC мһҗлҸҷ кё°мғҒ

**мғҒнҷ©**: н”Ңл Ҳмқҙм–ҙк°Җ мһҗлҠ” NPC(`isAwake: false`)м—җкІҢ л§җмқ„ кұҙлӢӨ

**л¬ём ң**: лҢҖнҷ”лҠ” м§„н–үлҗҳм§Җл§Ң `dailyPlan`мқҙ м—Ҷм–ҙм„ң лҢҖнҷ” мў…лЈҢ нӣ„ NPCк°Җ л©Ҳм¶Ө

**н•ҙкІ°**:
```
startConversation() нҳём¶ң мӢң:
  if (!isAwake) вҶ’ wakeUp() вҶ’ н•ҳлЈЁ кі„нҡҚ мғқм„ұ вҶ’ лҢҖнҷ” м§„н–ү
```

**кө¬нҳ„**: `npc-controller.ts:195-201`

### 2. лҢҖнҷ” мӨ‘ мқјм • мӢңк°„ мҙҲкіј к°җм§Җ

**мғҒнҷ©**: лҢҖнҷ”н•ҳлӢӨк°Җ лӢӨмқҢ мқјм • мӢңк°„мқ„ л„ҳкІЁлІ„лҰј (нҳ„мһ¬ 11:15, лӢӨмқҢ мқјм • 11:00)

**л¬ём ң**: `minutesUntilNext = -15`мқёлҚ° мһҗм • л„ҳк№Җ(23:00вҶ’01:00)кіј кө¬л¶„ н•„мҡ”

**н•ҙкІ°**:
```
if (minutesUntilNext < 0 && minutesUntilNext > -12*60):
  вҶ’ мӢӨм ң мқјм • мҙҲкіј! мҰүмӢң лҢҖнҷ” мў…лЈҢ
  вҶ’ thought: "мқҙлҹ°, 11:00м—җ м җмӢ¬ мӨҖл№„н•ҙм•ј н•ҳлҠ”лҚ° мқҙлҜё м§ҖлӮ¬лӢӨ!"
  вҶ’ л°ңнҷ”: "м•—, мқҙлҹ°! лІҢмҚЁ мқҙ мӢңк°„мқҙл„Ө. м җмӢ¬ мӨҖл№„ н•ҙм•ј н•ҙм„ң лЁјм Җ к°ҖліјкІҢ!"

if (minutesUntilNext < 0 && minutesUntilNext <= -12*60):
  вҶ’ мһҗм • л„ҳк№Җ (23:00 вҶ’ 01:00)
  вҶ’ minutesUntilNext += 24*60
```

**кө¬нҳ„**: `agent.ts:1312-1326`

### 3. лҢҖнҷ” мў…лЈҢ нӣ„ мһ¬н”ҢлһҳлӢқ (м§„м§ң Replanning)

**мғҒнҷ©**: кёҙ лҢҖнҷ”лЎң мғҒнҷ©мқҙ л°”лҖң (мҳҲ: "мҳӨлҠҳ лҸ„м Ғмқҙ лӮҳнғҖлӮ¬лҢҖмҡ”" м •ліҙ нҡҚл“қ)

**н•ҙкІ°**:
```
endConversation() мӢң:
  if (лҢҖнҷ” мӢңк°„ >= 30л¶„):
    вҶ’ replan(currentTime) нҳём¶ң  вҶҗ LLMмңјлЎң мғҲ кі„нҡҚ мғқм„ұ!
    вҶ’ мөңк·ј мӨ‘мҡ” м •ліҙ (importance >= 6) л°ҳмҳҒ
    вҶ’ нҳ„мһ¬ мӢңк°„л¶Җн„° 22:00к№Ңм§Җ мғҲ кі„нҡҚ
    вҶ’ кё°мЎҙ мҷ„лЈҢ кі„нҡҚ + мғҲ кі„нҡҚ лі‘н•©
```

**кё°мЎҙ кі„нҡҚ vs мһ¬н”ҢлһҳлӢқ**:
```
кё°мЎҙ: 10:00 м җмӢ¬мӨҖл№„ вҶ’ 12:00 мҶҗлӢҳл§һмқҙ вҶ’ 14:00 мІӯмҶҢ
           вҶ“ лҢҖнҷ”м—җм„ң "мҳӨлҠҳ мӢқмһ¬лЈҢ л°°лӢ¬ мҳЁлӢӨ" м •ліҙ
мһ¬н”ҢлһҳлӢқ: 12:00 мӢқмһ¬лЈҢ мҲҳл № вҶ’ 13:00 мһ¬кі  м •лҰ¬ вҶ’ 14:00 мҶҗлӢҳл§һмқҙ
```

**кө¬нҳ„**: `agent.ts:869-990`, `npc-controller.ts:242-260`

### 4. Thought л©”лӘЁлҰ¬ нғҖмһ…

**лӘ©м Ғ**: NPCмқҳ лӮҙл©ҙ нҢҗлӢЁ/нҳјмһЈл§җмқ„ кё°лЎқ (н”Ңл Ҳмқҙм–ҙм—җкІҢ ліҙмқҙм§Җ м•ҠмқҢ)

**л…јл¬ёкіј м°Ёмқҙ**: л…јл¬ём—җлҠ” observation/reflection/planл§Ң мһҲмқҢ

**мӮ¬мҡ© мӢңм җ**:
- лҢҖнҷ” м§ҖмҶҚ нҢҗлӢЁ мӢң: "л°”л№ м„ң лҢҖнҷ”лҘј к·ёл§Ңн•ҙм•јкІ лӢӨ"
- мһҗмңЁ л°ңнҷ” нҢҗлӢЁ мӢң: "м Җ мҶҗлӢҳм—җкІҢ л§җмқ„ кұём–ҙліјк№Ң?"
- мқјм • мҙҲкіј к°җм§Җ мӢң: "мқҙлҹ°, мӢңк°„мқҙ м§ҖлӮ¬лӢӨ!"

**кө¬нҳ„**: `agent.ts:addThought()`

**лӘ©м Ғ**: мһҗмңЁ л°ңнҷ” мҠӨнҢё л°©м§Җ (к°ҷмқҖ н”Ңл Ҳмқҙм–ҙм—җкІҢ л°ҳліө л§җ кұёкё° л°©м§Җ)

**м„Өм •**: 3мҙҲ мҝЁлӢӨмҡҙ
